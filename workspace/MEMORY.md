- 系统整改后自动同步 AGENTS.md + MEMORY.md + IDENTITY.md + TOOLS.md

## 系统架构
- V3.3: 波段(MX模拟 ¥50,000) + 打板(华泰模拟 ¥10,000) + 中枢分配
- 数据源: 腾讯+sina+mootdx(免费) | 60s缓存
- 调度: 37条cron | 06:00-22:30全覆盖
- 信号: 5维加权评分(趋势40+量能25+位置20+PE10+资金5) + 量价关系检测
- 过滤: 涨跌停+量比0.3x+日内高位85%+低位下跌 7道关卡
- 仓位: 评分分档 70+→¥600 / 55+→¥450 / <55→¥300
- 止盈: 市场自适应(BULL放宽/BEAR收紧) + 强趋势ATR跟踪
- 熔断: 仅跌停暴增(千股跌停级)

## 关键教训
- T+1下追突破风险极大，优先做回调买入
- 尾盘14:50是最安全的入场时段
- 东财API会限流，请求间隔需≥0.3秒
- 新浪概念API不可用，需东财或预设数据
- Clash代理在Global模式会干扰国内API
- Agent禁止主动交易，所有交易走cron
- hub_dispatcher不能逐只调mootdx(518只超时),必须用批量数据快速评分
- self_evolve遗传算法对¥20,000账户无意义,确定性规则更可靠
- 熔断4条规则无数据源,只剩跌停检测有效
- band_monitor防御主导必须exit,不能只打印

## 成功模式
- 加权信号(趋势40%)比等权6维更能区分赢家/输家
- 中枢分配+快速批量评分有效过滤垃圾标的
- 评分分档风险(¥300-600)比固定¥600更稳
- 市场自适应止盈(BULL/BEAR)比一刀切更优
- 量价关系检测可区分真拉vs诱多
- 日内高位过滤避免追高
- 跌停暴增熔断(千股级别)比复杂熔断更可靠

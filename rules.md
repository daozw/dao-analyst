# 交易系统 V3.2 (资金动向版)

## 核心指标
- 主力净流入: 趋势温度计 (偏多/偏空)
- 大单净额: 真买真卖确认 (大单买入-大单卖出)

## 策略矩阵

### 🚀 首板 (情绪型)
资金: 2000/笔
买入: 涨停 + 20日新高 + 换手>3% + 放量
止损: ①破首板涨停价 ②破MA10 ③主力连续2天为负 → 任一出局
止盈: ①主力3日<0且冲高回落→减50% ②到压力位+大单转负→减40% ③破MA5减→破MA10清
口诀: 破首板价/破MA10/主力3日转负→走; 主力3日转正才敢拿

### 📥 低吸 (趋势型)  
资金: 5000/笔 最多3只
买入: MA20↑ + 近20日破60日高 + 缩量 + 换手3-10% + 回撤≤8%
止损: ①破10日低点 ②破MA20且次日收不回 ③买入3天后主力仍负且站不回MA10
止盈: ①反弹MA10+主力3日<0→减仓 ②主力3日>0且>MA10→持有MA10移动止盈 ③到压力位+大单转负→落袋
加仓: 主力3日>0 且 收盘>MA10 → 加仓 (再启动信号)
口诀: 破10日低/破MA20/3天主力仍负→走; 主力转正+站MA10→才加

### 🛡️ V2.5 波段 (防守型)
资金: 2000/笔 最多3只
买入: PE<25 PB<2.5 涨幅3-8% 换手>2% 主板
止损: ①2xATR ②主力3日<0且破MA5
止盈: 追踪-5%从高点 ③主力3日转负+大单转负→先走

### 🔥 动量 (进攻型)
资金: 2000/笔
买入: MA5>MA20 + 近20日新高 + 放量>5日均量1.5倍 + 换手5-20%
止损: ①破MA5 ②主力2日连续为负
止盈: ①+10% ②大单净额转负→减半

## 防踩坑规则
股价涨+主力连续为负 → 拉高出货/诱多 → 不加仓 逢压力位减仓

## 仓位管理
- 双策略活跃: 全力2.2万 (低吸1.5+波段0.5+动量0.2)
- 单策略活跃: 半仓1万
- 双策略归零: 空仓
